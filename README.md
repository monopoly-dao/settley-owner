# Settley - Fractional Real Estate Ownership Platform

![Settley](https://settley.co/images/og.png)

## 🏠 Overview

**Settley** (formerly Monopoly DAO) is a revolutionary decentralized application (dApp) that enables fractional ownership of premium real estate properties worldwide. Built on blockchain technology, Settley democratizes property investment by allowing users to own shares of high-value properties for as low as $1, making real estate investment accessible to everyone.

The platform combines the transparency and security of blockchain with traditional real estate ownership through a legally-compliant tenancy-in-common structure, managed by Settley DAO LLC (Wyoming).

## 🚀 Getting Started

### Prerequisites
- Node.js 20.x or higher
- npm or yarn package manager
- MetaMask or compatible Web3 wallet
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-org/monopoly-dapp.git
   cd monopoly-dapp/dapp
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Set up environment variables**
   Create a `.env.local` file based on `.env.development`:
   ```env
   NEXT_PUBLIC_API_URL=your_api_url
   NEXT_PUBLIC_STRIPE_KEY=your_stripe_key
   NEXTAUTH_SECRET=your_nextauth_secret
   NEXTAUTH_URL=http://localhost:3000
   # Add other required environment variables
   ```

4. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Building for Production

```bash
npm run build
npm run start
```

### Linting

```bash
npm run lint
```
